# ESP32-S3---GPS-Flipper-Module
A compact Flipper Zero add-on that pairs an ESP32-S3 with an onboard GPS and a microSD slot purpose built for [Marauder](https://github.com/HacKitsuneDotCom/ESP32Marauder) firmware, with the flexibility to flash with custom firmware of your choice. Use the&nbsp;Flipper’s own screen and controls to navigate menus, run scans, and log results.

All boards are pre-flashed with latest ESP Merauder firmware, use the HacKitsune Web Flasher to update or change firmware.
